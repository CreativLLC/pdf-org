# Permission Set Groups

Both PSGs in this engagement are in the `force__` namespace, i.e., shipped by Salesforce-managed packages (Sales Workspace, Scale Center). Neither was authored by the engagement; both are listed here for completeness because their member permission sets are also `force__`-namespaced and not under engagement control.

## `force__SalesWorkspacePSG`

**Description**: _Not set._

**Composed permission sets**:
- `force__PipelineInspectionUser` — managed package; not under engagement control.
- `force__SalesWorkspace` — managed package; not under engagement control.

**Typical assignment**: _Not documented in source. Assigned in Setup → Permission Set Groups; consult Setup for the membership list._
